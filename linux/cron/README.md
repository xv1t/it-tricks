#CRON

Edit tasks in the default editor
```bash
crontab -e
```

Change text editor type `select-editor`
```bash
Select an editor.  To change later, run 'select-editor'.
  1. /bin/ed
  2. /bin/nano        <---- easiest
  3. /usr/bin/mcedit
  4. /usr/bin/vim.basic
  5. /usr/bin/vim.tiny
```

After change save file and exit
```
crontab: installing new crontab
```
